# JWAT-Tools #

JWAT-Tools is an extension to the JWAT utility libraries.

This project currently includes a commandline tool with various gzip/arc/warc/xml tasks.

### Tasks ###

* Arc2Warc
* CDX
* Changed
* Compress
* ContainerMD
* Decompress
* Delete
* Extract
* Interval
* PathIndex
* Test
* Unpack
